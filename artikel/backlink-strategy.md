# Backlink Strategy & Link Building — LINETOGEL

## Platform Gratis (COPY-PASTE READY)

### 1. Medium.com
- Buat akun Medium gratis
- Post artikel `01-panduan-situs-toto-terpercaya.md`
- Jangan terlalu banyak link — 2-3 link natural ke linetogel-site.vercel.app

### 2. Blogger.com (Blogspot)
- Buat blog baru dengan domain mengandung kata kunci
- Post `02-bayaran-tertinggi.html` sebagai postingan
- Edit template tambah meta description

### 3. WordPress.com
- Buat blog gratis
- Post artikel yang sama dengan sedikit modifikasi biar gak duplicate

### 4. Sosial Media Profiles
- Twitter/X: buat akun @linetogel, bio → link ke https://linetogel-site.vercel.app
- Facebook Page: buat halaman LINETOGEL, isi website
- Instagram: bio link
- TikTok: bio link
- Pinterest: buat pin yang link ke landing page

### 5. Forum & Komunitas
- Kaskus: buat thread di lounge/diskusi
- Reddit: r/indonesia atau subreddit terkait
- Quora: jawab pertanyaan tentang toto online, sisipkan link natural

### 6. Web Directory (Submit)
- idwebhost.com/directory
- direktori.web.id
- Submit ke direktori gratisan Indonesia

### 7. Guest Post / Web 2.0
- Buat blog di Weebly, Wix, Jimdo, Strikingly
- Masing-masing post 1 artikel dengan 1-2 link ke target

## Backlink Tier Structure
```
Tier 1 (Langsung ke linetogel-site.vercel.app):
  - Medium, Blogger, WordPress.com, Sosial Media

Tier 2 (Link ke Tier 1):
  - Web 2.0 (Weebly, Wix, dll)
  - Forum posts
  - Blog comments

Tier 3 (Link ke Tier 2):
  - Social bookmarks
  - Directory submissions
```
